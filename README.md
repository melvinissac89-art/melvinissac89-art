## 🚀 Current Projects

### 🎨 PaintMaster ERP

A Windows-first, offline-first business management platform designed specifically for paint dealers.

- Product catalogue and variant management
- Inventory and multi-location stock control
- Batches, stock movements, transfers, reservations, and adjustments
- Stock counts, reorder monitoring, valuation, and reports
- Authentication, role-based access control, sessions, and audit logging
- Packaged Windows runtime and automated CI verification

**Stack:** Electron, React, TypeScript, Vite, Tailwind CSS, Fastify, Prisma, SQLite, Vitest

### 🧭 AMC Nexus

A multi-tenant SaaS platform for job discovery, extraction, candidate workflows, review operations, and automation.

- Organization-level tenant isolation
- Candidate and operator workspaces
- Job discovery and extraction pipelines
- Review queues and workflow execution
- Scheduling and background task processing
- Role-based permissions and audit trails

**Stack:** React, TypeScript, Python, FastAPI, SQLAlchemy, Alembic, PostgreSQL

### 👶 Daycare Management Platform

A role-based management system connecting administrators, teachers, and parents.

- Attendance and calendar workflows
- Parent and teacher dashboards
- Messaging and notifications
- User profiles and authentication
- Role-based administration
- Mobile application workflows

**Stack:** React, Node.js, Prisma, Flutter, REST APIs

### 📈 Trading Structure Engine

A market-structure analysis engine for identifying price-action events in financial chart data.

- Break of Structure detection
- Change of Character detection
- Liquidity Sweep identification
- Interactive chart visualization
- Historical event analysis and validation

**Stack:** Python, financial time-series data, Plotly, Lightweight Charts
